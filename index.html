Technical Projects

Naïve Bayes Text Classification

This program uses the Naive Bayes method to learn a classifier for text from training data, and using a test set to evaluate the quality of the classifier. The input file consists of short biographies of people tagged with a category, and the program outputs the probabilities associated with each category for each person in the test set, along with the prediction and whether it is right or wrong, and the overall accuracy of the classifier on the test set. The program is divided into two parts, learning the classifier from the training set and evaluating the classifier over the test set.
Technology used:

  • Python
  • Supervised machine learning
  • Naïve Bayes Algorithm

Game of Chance

A program that learns to play a simple game of chance similar to Blackjack. The program learns from a series of games played against itself, keeping track of the number of wins and losses for each combination of player scores and number of dice rolled. The goal is to maximize the number of wins by choosing the optimal number of dice to roll on each turn based on the current score.
Technology used:

  • Python
  • AI

Word Count Wizard

Word Count Wizard counts the number of unique words in the text provided by the user, returns the frequency of any word in that given text among other operations. The wizard takes the input text and organize it in a hash table using 4 different hash functions implementing Summation, Polynomial, Cyclic Shift, and Summation with MAD compression.
Technology used:

  • C
  • Hash Tables
